РПомощьСловарь.

  • Программа имеет встроенный локальный словарь, примерно 50 000 слов
    Основные слова не имеют транскрипцию, ( можете редактировать сами)
    Можно вносить новые слова, правило ввода новых слов:

    Редактрование, добавление слов
    Например: Runner - ключевое слов
    Слово значение:Rynner Бегун ( через пробел)
    Так же, при редактировании, значение должно повторять ключевое слово и перевод через пробел.
    Ввоод поиска осуществляется как английской транскрипцией так и русской.
    Переключение из настройки программы.
    Если слово не присутствует в словаре, программа обращается к Yndex транслятору ( если есть Интернет).
    условия запроса к Yandex в настройках программы

  • Перевод текстов.
    Текст перевода вы можете добавлять сами в паку linqion/data/ в корневой директории вашего девайса.
    Расширение файлов для перевода: .txt, для учебной информации: .html, кодировка utf8 ( очень важно)

  • Локальный словарь:
    Временный словарь - избранные, служит для лучшего запоминания слов и для улучшения перевода текстов.
